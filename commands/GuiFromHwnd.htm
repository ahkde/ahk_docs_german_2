GuiFromHwnd

Ermittelt das Gui-Objekt eines GUI-Fensters, das zu einer bestimmten HWND-Nummer gehört.

GuiObj := GuiFromHwnd(Hwnd , ÜbergeordnetesFenster)

Parameter

Hwnd

Typ: Integer

Die HWND-Nummer (Fenster-Handle) eines GUI-Steuerelements, das zuvor vom Skript erstellt wurde, oder, wenn ÜbergeordnetesFenster True ist, ein beliebiges untergeordnetes Fenster eines vom Skript erstellten GUI-Steuerelements.

ÜbergeordnetesFenster

Typ: Boolescher Wert

Wenn dieser Parameter True ist und Hwnd ein untergeordnetes Fenster identifiziert, das keine GUI ist, wird die Funktion das nächstgelegene übergeordnete Fenster, das eine GUI ist, suchen und abrufen. Ansonsten gibt die Funktion eine leere Zeichenkette zurück, wenn Hwnd nicht direkt ein GUI-Fenster identifiziert.

Rückgabewert

Typ: Gui oder Zeichenkette (leer)

Diese Funktion gibt das Gui-Objekt zurück, das mit der angegebenen HWND-Nummer verknüpft ist, oder eine leere Zeichenkette, wenn es kein Objekt gibt oder die HWND-Nummer ungültig ist.

Bemerkungen

Zum Beispiel kann die HWND-Nummer eines GUI-Fensters an eine OnMessage-Funktion übergeben oder über Gui.Hwnd, WinExist oder eine andere Methode abgerufen werden.

Beispiele

Ermittelt das Gui-Objekt über die HWND-Nummer des gerade erstellten GUI-Fensters und meldet dessen Titel.

MeineGui := Gui(, "Fenstertitel")
MeineGui.Add("Text",, "Beliebiger Text.")
MeineGui.Show()

MsgBox(GuiFromHwnd(MeineGui.Hwnd).Title)
