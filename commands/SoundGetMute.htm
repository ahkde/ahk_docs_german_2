SoundGetMute

Ruft die Stummschalteinstellung eines Audiogeräts ab.

Einstellung := SoundGetMute(Komponente, Gerät)

Parameter

Komponente

Typ: Zeichenkette oder Integer

Anzeigename und/oder Indexnummer der Komponente. Zum Beispiel 1, "Line in" oder "Line in:2".

Wenn dieser Parameter weggelassen wird oder leer ist, wird die Master-Stummschalteinstellung abgerufen.

Weitere Informationen finden Sie unter Komponente (Sound-Funktionen).

Gerät

Typ: Zeichenkette oder Integer

Anzeigename und/oder Indexnummer des Geräts. Zum Beispiel 1, "Lautsprecher", "Lautsprecher:2" oder "Lautsprecher (Beispiel HD Audio)".

Lässt man diesen Parameter weg, wird standardmäßig das als Standard festgelegte Wiedergabegerät des Systems verwendet (was nicht unbedingt Gerät 1 ist).

Weitere Informationen finden Sie unter Gerät (Sound-Funktionen).

Rückgabewert

Typ: Integer (boolescher Wert) oder Zeichenkette (leer)

Diese Funktion gibt 0 (nicht stummgeschaltet) oder 1 (stummgeschaltet) zurück.

Fehlerbehandlung

Bei Misserfolg kann eine der folgenden Ausnahmen ausgelöst werden:

Error-KlasseMessage
TargetErrorDevice not found (Gerät nicht gefunden)
Component not found (Komponente nicht gefunden)
Component doesn't support this control type (Komponente unterstützt diesen Regler nicht)
OSErrorVom System generiert.

Bemerkungen

Um den Leistungsumfang der installierten Audiogeräte auf dem Betriebssystem zu ermitteln -- z. B. die Namen und verfügbaren Komponenten -- können Sie das Soundkartenanalyse-Skript ausführen.

Beispiele

Überprüft, ob das als Standard festgelegte Wiedergabegerät stummgeschaltet ist.

master_mute := SoundGetMute()
if master_mute
    MsgBox "Das als Standard festgelegte Wiedergabegerät ist stummgeschaltet."
else
    MsgBox "Das als Standard festgelegte Wiedergabegerät ist nicht stummgeschaltet."

Überprüft, ob Line-In-Pass-Through stummgeschaltet ist.

if SoundGetMute("Line In") = 0
    MsgBox "Line-In-Pass-Through ist nicht stummgeschaltet."

Überprüft, ob das Mikrofon (Aufnahme) stummgeschaltet ist.

if SoundGetMute( , "Mikrofon") = 0
    MsgBox "Das Mikrofon (Aufnahme) ist nicht stummgeschaltet."
