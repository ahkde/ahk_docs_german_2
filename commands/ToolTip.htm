ToolTip

Erstellt ein Fenster irgendwo auf dem Bildschirm, das immer im Vordergrund ist.

ToolTip Text, X, Y, WelchesToolTip

Parameter

Text

Typ: Zeichenkette

Wenn dieser Parameter weggelassen wird oder leer ist, wird das ToolTip (sofern vorhanden) versteckt. Ansonsten kann eine Zeichenkette angegeben werden, die im ToolTip angezeigt werden soll. Um ein ToolTip mehrzeilig zu machen, trennen Sie jede Zeile mit einem Zeilenumbruchszeichen (`n), z. B. Zeile1`nZeile2.

Wenn der Text relativ lang ist, kann er via Fortsetzungsbereich in mehreren kurzen Zeilen aufgeteilt werden, um die Übersichtlichkeit und Wartbarkeit des Skripts zu erhöhen.

X, Y

Typ: Integer

Die X- und Y-Koordinate der oberen linken Ecke des Tooltips, relativ zum aktiven Fenster (mit CoordMode "ToolTip" können die Koordinaten relativ zum Bildschirm gemacht werden). Lässt man die Koordinaten weg, wird das ToolTip in der Nähe des Mauszeigers angezeigt.

WelchesToolTip

Typ: Integer

Dieser Parameter dient dazu, mehrere ToolTips gleichzeitig anzeigen zu lassen. Eine Nummer zwischen 1 und 20 kennzeichnet, auf welchem ToolTip-Fenster die Operation durchgeführt werden soll. Wenn keine Nummer angegeben ist, wird standardmäßig 1 verwendet (das erste Fenster).

Bemerkungen

Ein typisches ToolTip sieht wie folgt aus:

Befindet sich die angegebene X- und Y-Koordinate außerhalb des Bildschirms, wird das ToolTip neu positioniert, um vollständig sichtbar zu bleiben.

Das ToolTip wird solange angezeigt, bis eine der folgenden Bedingungen stattfindet:

  • Das Skript wird terminiert.
  • Die ToolTip-Funktion wird mit leerem Text-Parameter erneut ausgeführt.
  • Der Benutzer hat das ToolTip angeklickt (dieses Verhalten variiert je nach Betriebssystem).

Mithilfe der OwnDialogs-Option kann ein ToolTip einem GUI-Fenster untergeordnet werden. ToolTips dieser Art werden automatisch zerstört, sobald ihr übergeordnetes Fenster zerstört wird.

Beispiele

Zeigt ein mehrzeiliges Tooltip an einer bestimmten Position im aktiven Fenster an.

ToolTip "Mehrzeiliges`nToolTip", 100, 150

Versteckt ein Tooltip nach einer gewissen Zeit, ohne Sleep verwenden zu müssen (das den aktuellen Thread anhalten würde).

ToolTip "Zeitgesteuertes ToolTip`nEs wird 5 Sekunden lang angezeigt."
SetTimer () => ToolTip(), -5000
