Shutdown

Kann das System herunterfahren, neustarten oder abmelden.

Shutdown Flag

Parameter

Flag

Typ: Integer

Eine Kombination (Summe) der folgenden Zahlen:

  • 0 = Abmelden
  • 1 = Herunterfahren
  • 2 = Neustarten
  • 4 = Erzwingen
  • 8 = Ausschalten

Addieren Sie die gewünschten Werte. Zum Beispiel ist zum Herunterfahren und Ausschalten des Systems eine 9 notwendig (Herunterfahren + Ausschalten = 1 + 8 = 9).

Der "Erzwingen"-Wert (4) zwingt alle offenen Programme sich zu schließen. Diese Methode sollte nur im Notfall verwendet werden, weil offene Programme dadurch Daten verlieren könnten.

Der "Ausschalten"-Wert (8) fährt das System herunter und schaltet den Strom aus.

Bemerkungen

Um das System anzuhalten oder in den Ruhemodus zu versetzen, schauen Sie sich Beispiel #2 am unteren Ende dieser Seite an.

Um den Monitor auszuschalten, schauen Sie sich SendMessage-Beispiel #1 an.

Des Weiteren kann mithilfe von OnExit ein Skript dazu gebracht werden, zu erkennen, wann das System herunterfährt oder der Benutzer sich abmeldet.

Siehe auch

Run, ExitApp, OnExit

Beispiele

Erzwingt einen Neustart (Neustart + Erzwingen = 2 + 4 = 6).

Shutdown 6

Ruft die API-Funktion "SetSuspendState" von Windows auf, um das System anzuhalten oder in den Ruhemodus zu versetzen. Beachten Sie, dass der zweite Parameter auf neueren Systemen möglicherweise überhaupt keine Wirkung hat.

; Parameter #1: Übergeben Sie 1 statt 0, um Ruhezustand statt Anhalten zu bewirken.
; Parameter #2: Übergeben Sie 1 statt 0, um Anhalten ohne Nachfrage zu erzwingen.
; Parameter #3: Übergeben Sie 1 statt 0, um alle Aufwachereignisse zu deaktivieren.
DllCall("PowrProf\SetSuspendState", "Int", 0, "Int", 0, "Int", 0)
