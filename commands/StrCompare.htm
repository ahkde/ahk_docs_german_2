StrCompare

Vergleicht zwei Zeichenketten alphabetisch.

Ergebnis := StrCompare(Zkette1, Zkette2 , GroßKleinSensitiv)

Parameter

Zkette1, Zkette2

Typ: Zeichenkette

Zeichenketten, die verglichen werden sollen.

GroßKleinSensitiv

Typ: Integer oder Zeichenkette

Geben Sie einen der folgenden Werte an (standardmäßig 0, wenn dieser Parameter weggelassen wird):

"On" oder 1 (True): Der Vergleich erfolgt Groß-/Kleinschreibung-sensitiv.

"Off" oder 0 (False): Die Großbuchstaben A bis Z und die entsprechenden Kleinbuchstaben werden gleichwertig behandelt.

"Locale": Der Vergleich erfolgt nicht Groß-/Kleinschreibung-sensitiv, gemäß den Regeln der aktuellen Sprach- und Regionseinstellungen des Benutzers. In den meisten englischen und westeuropäischen Regionen beispielsweise werden die Großbuchstaben A bis Z, einschließlich ASCII-fremde Zeichen wie Ä und Ü, und die entsprechenden Kleinbuchstaben gleichwertig behandelt. Locale ist 1- bis 8-mal langsamer als Off, abhängig von den zu vergleichenden Zeichenketten.

"Logical": Wie Locale, aber die Ziffern in den Zeichenketten werden nicht als Text, sondern als numerischer Inhalt behandelt. Zum Beispiel wird "A2" so behandelt, als wäre es kleiner als "A10". Wenn sich jedoch zwei Zahlen nur durch das Vorhandensein einer Null am Anfang unterscheiden, wird die Zeichenkette mit einer Null am Anfang möglicherweise so behandelt, als wäre sie kleiner als die andere Zeichenkette. Das genaue Verhalten kann je nach Version des Betriebssystems variieren.

Rückgabewert

Typ: Integer

Um die Beziehung zwischen Zkette1 und Zkette2 deutlich zu machen, gibt diese Funktion einen der folgenden Werte zurück:

  • 0, wenn Zkette1 identisch zu Zkette2 ist
  • einen positiven Integer, wenn Zkette1 größer als Zkette2 ist
  • einen negativen Integer, wenn Zkette1 kleiner als Zkette2 ist

Um eine bestimmte Beziehung zwischen den beiden Zeichenketten zu überprüfen, vergleichen Sie das Ergebnis mit 0. Zum Beispiel:

a_kleiner_als_b := StrCompare(a, b) < 0
a_größer_gleich_b := StrCompare(a, b) >= 0

Bemerkungen

Diese Funktion wird hauptsächlich für Sort-Callbacks verwendet.

Siehe auch

Sort, VerCompare

Beispiele

Demonstriert den Unterschied zwischen einem nicht-Groß-/Kleinschreibung-sensitiven und Groß-/Kleinschreibung-sensitiven Vergleich.

MsgBox StrCompare("Abc", "abc") ; Gibt 0 zurück
MsgBox StrCompare("Abc", "abc", true) ; Gibt -1 zurück
