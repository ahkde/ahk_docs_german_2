SetCapsLockState / SetNumLockState / SetScrollLockState

Setzt den Status von Feststell/Num/Rollen. Kann auch das An- oder Ausbleiben der Taste erzwingen.

SetCapsLockState Status
SetNumLockState Status
SetScrollLockState Status

Parameter

Status

Typ: Zeichenkette oder Integer (boolescher Wert)

Lässt man diesen Parameter weg, wird das AlwaysOn/Off-Attribut von der Taste entfernt (falls vorhanden). Ansonsten kann eines der folgenden Wörter angegeben werden:

On oder 1 (true): Schaltet die Taste ein und entfernt ihr AlwaysOn/Off-Attribut (falls vorhanden).

Off oder 0 (false): Schaltet die Taste aus und entfernt ihr AlwaysOn/Off-Attribut (falls vorhanden).

AlwaysOn: Erzwingt den eingeschalteten Zustand der Taste.

AlwaysOff: Erzwingt den ausgeschalteten Zustand der Taste.

Bemerkungen

Alternativ zu Beispiel #3 unten kann eine Taste auch via Send-Funktion in ihrem entgegengesetzten Zustand umgeschaltet werden; zum Beispiel: Send "{CapsLock}". Allerdings benötigt das Senden von {CapsLock} möglicherweise vorher SetStoreCapsLockMode False.

Damit eine Taste AlwaysOn oder AlwaysOff bleibt, ist der Tastatur-Hook erforderlich, der in solchen Fällen automatisch installiert wird.

Siehe auch

SetStoreCapsLockMode, GetKeyState

Beispiele

Schaltet Num ein und entfernt den AlwaysOn/Off-Attribut (falls vorhanden).

SetNumLockState True

Erzwingt den ausgeschalteten Zustand von Rollen.

SetScrollLockState "AlwaysOff"

Schaltet Feststell in den entgegengesetzten Zustand.

SetCapsLockState !GetKeyState("CapsLock", "T")
