IsLabel

Gibt eine Nummer ungleich 0 zurück, wenn ein bestimmtes Label im aktuellen Gültigkeitsbereich existiert.

IstLabel := IsLabel(LabelName)

Parameter

LabelName

Typ: Zeichenkette

Der Name eines Labels.

Rückgabewert

Typ: Integer (boolescher Wert)

Diese Funktion gibt 1 (true) zurück, wenn das angegebene Label innerhalb des aktuellen Gültigkeitsbereichs existiert, oder 0 (false), wenn nicht.

Bemerkungen

Diese Funktion ist nützlich, um Laufzeitfehler beim Spezifizieren eines dynamischen Labels für Goto zu vermeiden.

Wenn sie innerhalb einer Funktion aufgerufen wird, werden nur die Labels dieser Funktion überprüft. Globale Labels sind keine gültigen Ziele für ein lokales Goto.

Meldet "Ziellabel existiert", da das Label tatsächlich existiert.
