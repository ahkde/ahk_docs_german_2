AutoHotkey v2

AutoHotkey v2 soll die Benutzerfreundlichkeit und Einfachheit der Sprache und des Befehlssatzes verbessern, ohne die Abwärtskompatibilität zu berücksichtigen. Daher werden Skripte, die für v1 geschrieben wurden, grundsätzlich nicht ohne Änderungen mit v2 funktionieren. Die Syntax ist insgesamt konsistenter, mit deutlich weniger Macken und Fallen, außerdem wurden viele Verbesserungen vorgenommen.

Ausführliche Details zu den Unterschieden zwischen v1 und v2 finden Sie unter Änderung von v1.1 bis v2.0.

Aktueller Stand: AutoHotkey v2.0-rc.1 wurde am 20.11.2022 veröffentlicht. Der finale v2.0.0-Release ist für Ende 2022 geplant, allerdings sind alle geplanten Features und Änderungen am Code bereits implementiert, daher liegt der Schwerpunkt der Entwicklung nun auf Qualitätskontrolle, Paketierung und Dokumentation.

Anforderungen: Windows 7 und höher werden unterstützt.

Installation

v2.0-beta.4 und höher können zusammen mit v1.1 installiert werden, so dass Skripte, die für beide Versionen geschrieben wurden, ohne großen Aufwand funktionieren. Einzelheiten finden Sie in der Dokumentation unter Installation und Launcher.

Um die Installation mit der EXE-Datei durchzuführen, laden Sie sie einfach herunter, führen Sie sie aus, prüfen Sie die Optionen und klicken Sie auf Install.

Um die Installation mit der ZIP-Datei durchzuführen:

  1. Öffnen Sie die ZIP-Datei mit dem Dateiexplorer (kein Entpacken erforderlich), oder entpacken Sie den Inhalt der ZIP-Datei in ein temporäres Verzeichnis.
  2. Führen Sie Install.cmd aus.
  3. Wenn Sie eine Meldung wie "Der Herausgeber konnte nicht verifiziert werden. Möchten Sie diese Software ausführen?" erhalten, klicken Sie auf Ausführen.
  4. Klicken Sie auf Install (nachdem Sie ggf. die Optionen geändert haben).

Nach der Installation öffnet sich Dash automatisch. Dies wird sich voraussichtlich vor der Veröffentlichung von v2.0.0 ändern.
