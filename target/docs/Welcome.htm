Erste Schritte

Bevor AutoHotkey etwas tun kann, müssen Sie ein Skript schreiben, damit AutoHotkey weiß, was es tun soll.
Um zu erfahren, wie das geht, können Sie sich das Tutorial anschauen oder direkt damit anfangen.

Grundsätzlich gilt: erstelle eine Skriptdatei → füge einige Hotkeys hinzu → führe die Skriptdatei aus.

In unserem freundlichen Diskussionsforum können Sie Hilfe und viele nützliche Skripte finden.

Warum sehe ich diese Seite?

Sie haben AutoHotkey gestartet, ohne dem Programm zu sagen, welche Skriptdatei es ausführen soll, außerdem konnte nirgendwo ein Skript an den üblichen Stellen gefunden werden.
