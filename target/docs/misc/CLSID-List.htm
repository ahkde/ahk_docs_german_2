CLSID-Liste (Fensterklassenidentifikatoren)

Bestimmte spezielle Verzeichnisse im Betriebssystem werden durch eindeutige Zeichenketten identifiziert. Einige von diesen Zeichenketten können in Verbindung mit FileSelect, DirSelect und Run verwendet werden. Zum Beispiel:

AusgabeVar := FileSelect(, "::{645ff040-5081-101b-9f08-00aa002f954e}")  ; Wählt eine Datei im Papierkorb aus.
AusgabeVar := DirSelect("::{20d04fe0-3aea-1069-a2d8-08002b30309d}")  ; Wählt einen Ordner im Arbeitsplatz/Computer aus.

Die "Ja"-Einträge in der letzten Spalte sind nicht verbindlich: die Run-Funktion unterstützt je nach Systemkonfiguration möglicherweise unterschiedliche CLSIDs. Um einen CLSID-Ordner via Run zu öffnen, geben Sie einfach als ersten Parameter eine CLSID-Zeichenkette an. Zum Beispiel:

Run "::{20d04fe0-3aea-1069-a2d8-08002b30309d}"  ; Öffnet den Arbeitsplatz/Computer.
Run "::{645ff040-5081-101b-9f08-00aa002f954e}"  ; Öffnet den Papierkorb.
Run "::{450d8fba-ad25-11d0-98a8-0800361b1103}\Mein Ordner"  ; Öffnet einen Ordner in "Eigene Dokumente".
Run A_MyDocuments "\Mein Ordner"  ; Wie oben, funktioniert aber in den meisten Systemen.
