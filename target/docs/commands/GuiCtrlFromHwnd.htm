GuiCtrlFromHwnd

Ermittelt das GuiControl-Objekt eines GUI-Steuerelements, das zu einer bestimmten HWND-Nummer gehört.

GuiControlObj := GuiCtrlFromHwnd(Hwnd)

Parameter

Hwnd

Typ: Integer

Die HWND-Nummer (Fenster-Handle) eines GUI-Steuerelements oder eines Fensters, das einem solchen Steuerelement untergeordnet ist (z. B. das Edit-Steuerelement einer ComboBox). Das Steuerelement muss vom aktuellen Skript erstellt worden sein, also durch Aufrufen von Add oder AddSteuerelement.

Rückgabewert

Typ: GuiControl oder Zeichenkette (leer)

Diese Funktion gibt das GuiControl-Objekt zurück, das mit der angegebenen HWND-Nummer verknüpft ist, oder eine leere Zeichenkette, wenn es kein Objekt gibt oder die HWND-Nummer ungültig ist.

Bemerkungen

Zum Beispiel kann die HWND-Nummer (eindeutige ID) eines GUI-Steuerelements via GuiCtrl.Hwnd, MouseGetPos oder OnMessage abgerufen werden.
