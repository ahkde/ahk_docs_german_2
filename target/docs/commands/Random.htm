Random

Erzeugt eine pseudozufällige Zahl.

N := Random(A, B)

Parameter

A, B

Typ: Integer oder Float

Die minimale und/oder maximale Zahl, die erzeugt werden kann, in beliebiger Reihenfolge. Wenn nur ein Parameter angegeben ist, wird für den anderen Parameter standardmäßig 0 verwendet. Lässt man beide Parameter weg, wird standardmäßig 0.0 und 1.0 verwendet.

Handelt es sich um Integer, sind sowohl der Minimalwert als auch der Maximalwert Bestandteil der möglichen Zahlen, die zurückgegeben werden können. Es wird der gesamte 64-Bit-Integer-Bereich unterstützt.

Handelt es sich um Floating-Point-Zahlen, ist der Maximalwert generell ausgeschlossen.

Rückgabewert

Typ: Integer oder Float

Diese Funktion erzeugt eine pseudozufällige Zahl - also eine Zahl, die eine echte Zufallszahl vortäuscht, aber in Wirklichkeit auf Basis einer komplizierten Formel errechnet wurde, um das Bestimmen/Erraten der nächsten Zahl extrem zu erschweren.

Wenn A oder B eine Floating-Point-Zahl ist oder beide weggelassen werden, wird das Ergebnis eine Floating-Point-Zahl sein. Ansonsten wird das Ergebnis ein Integer sein.

Bemerkungen

Jede Zahl innerhalb des angegebenen Bereichs hat ungefähr die gleich hohe Chance, generiert zu werden.

Obwohl der angegebene Maximalwert designbedingt ausgeschlossen wird, kann er theoretisch aufgrund von Floating-Point-Rundungsfehlern zurückgegeben werden. Dies wurde nicht bestätigt und ist möglicherweise nur möglich, wenn die gewählten Grenzen größer als 2**53 sind. Beachten Sie auch, dass angesichts der Tatsache, dass es bis zu 2**53 mögliche Werte geben kann (z. B. im Bereich von 0.0 bis 1.0), die Chance sehr gering ist, genau die untere Grenze zu erzeugen.

Beispiele

Generiert einen zufälligen Integer im Bereich von 1 bis 10 und speichert ihn in N.

N := Random(1, 10)

Generiert einen zufälligen Integer im Bereich von 0 bis 9 und speichert ihn in N.

N := Random(9)

Generiert eine zufällige Floating-Point-Zahl im Bereich von 0.0 bis 1.0 und speichert sie in Bruch.

Bruch := Random(0.0, 1.0)
Bruch := Random()  ; Das gleiche wie oben.
