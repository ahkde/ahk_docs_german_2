SoundGetName

Ruft den Namen eines Audiogeräts oder einer Komponente ab.

Name := SoundGetName(Komponente, Gerät)

Parameter

Komponente

Typ: Zeichenkette oder Integer

Anzeigename und/oder Indexnummer der Komponente. Zum Beispiel 1, "Line in" oder "Line in:2".

Wenn dieser Parameter weggelassen wird oder leer ist, wird der Name des Geräts selbst abgerufen.

Weitere Informationen finden Sie unter Komponente (Sound-Funktionen).

Gerät

Typ: Zeichenkette oder Integer

Anzeigename und/oder Indexnummer des Geräts. Zum Beispiel 1, "Lautsprecher", "Lautsprecher:2" oder "Lautsprecher (Beispiel HD Audio)".

Lässt man diesen Parameter weg, wird standardmäßig das als Standard festgelegte Wiedergabegerät des Systems verwendet (was nicht unbedingt Gerät 1 ist).

Weitere Informationen finden Sie unter Gerät (Sound-Funktionen).

Rückgabewert

Typ: Zeichenkette

Der Name des Geräts oder der Komponente, der leer sein kann.

Fehlerbehandlung

Bei Misserfolg kann eine der folgenden Ausnahmen ausgelöst werden:

Error-KlasseMessage
TargetErrorDevice not found (Gerät nicht gefunden)
Component not found (Komponente nicht gefunden)
OSErrorVom System generiert.

Beispiele

Ermittelt und meldet den Namen des als Standard festgelegten Wiedergabegeräts.

default_device := SoundGetName()
MsgBox 'Das als Standard festgelegte Wiedergabegerät ist "' default_device '".'

Ermittelt und meldet den Namen des ersten Geräts.

device1 := SoundGetName( , 1)
MsgBox 'Gerät 1 ist "' device1 '".'

Ermittelt und meldet den Namen der ersten Komponente.

component1 := SoundGetName(1)
MsgBox 'Komponente 1 ist "' component1 '".'

Ein komplexeres Beispiel finden Sie unter Soundkartenanalyse-Skript.
