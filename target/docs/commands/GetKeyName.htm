GetKeyName

Ermittelt den Namen oder Text einer Taste.

Name := GetKeyName(TasteName)

Parameter

TasteName

Typ: Zeichenkette

Nahezu jedes Zeichen, das mit der Tastatur erzeugt werden kann, oder einer der Tastennamen aus der Tastenliste. Beispiele: B, 5, LWin, RControl, Alt, Enter, Escape.

Alternativ kann dieser Parameter ein expliziter virtueller Tastencode wie vkFF, ein expliziter Scancode wie sc01D oder eine Kombination aus VK und SC (in dieser Reihenfolge) wie vk1Bsc001 sein. Beachten Sie, dass diese Codes hexadezimal sein m端ssen.
