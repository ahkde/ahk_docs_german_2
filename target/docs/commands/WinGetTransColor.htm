WinGetTransColor

Gibt die Farbe zurück, die in einem bestimmten Fenster als transparent markiert wurde.

TransFarbe := WinGetTransColor(FensterTitel, FensterText, IgnoriereTitel, IgnoriereText)

Parameter

FensterTitel

Typ: Zeichenkette, Integer oder Objekt

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter FensterTitel-Parameter.

FensterText

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

IgnoriereTitel

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Titeln enthalten, werden ignoriert.

IgnoriereText

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Textelementen enthalten, werden ignoriert.

Rückgabewert

Typ: Zeichenkette

Diese Funktion gibt die 6-stellige RGB-Farbe zurück, die im angegebenen Fenster als transparent markiert wurde (siehe WinSetTransColor, wie eine transparente Farbe gesetzt werden kann). Der Rückgabewert ist eine leere Zeichenkette, wenn: 1) keine passenden Fenster gefunden werden; 2) das Fenster keine transparente Farbe hat; oder 3) andere Bedingungen auftreten (durch das Verhalten des Betriebssystems), z. B. wenn das transparent gemachte Fenster minimiert, wiederhergestellt und/oder skaliert wurde.

Fehlerbehandlung

Es wird ein TargetError ausgelöst, wenn das Fenster nicht gefunden werden konnte.

Bemerkungen

Die ID des Fensters, das sich unter dem Mauszeiger befindet, kann mit MouseGetPos abgerufen werden.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Siehe auch

WinSetTransColor, WinGetTransparent, Win-Funktionen, Control-Funktionen

Beispiele

Ruft die transparente Farbe eines Fensters unterhalb des Mauszeigers ab.

MouseGetPos ,, &MausFenster
TransFarbe := WinGetTransColor(MausFenster)
