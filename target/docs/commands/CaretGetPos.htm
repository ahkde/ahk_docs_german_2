CaretGetPos

Gibt die aktuelle Position des Textcursors (Texteinfügemarke) zurück.

TextcursorGefunden := CaretGetPos(&AusgabeVarX, &AusgabeVarY)

Parameter

&AusgabeVarX, &AusgabeVarY

Typ: VarRef

Referenzen, die auf Variablen verweisen, in denen die X- und Y-Koordinate gespeichert werden sollen. Standardmäßig sind die ermittelten Koordinaten relativ zum Clientbereich des aktiven Fensters, was aber mit CoordMode oder A_CoordModeCaret geändert werden kann.

Rückgabewert

Typ: Integer (boolescher Wert)

Wenn kein Fenster aktiv ist oder die Position des Textcursors nicht ermittelt werden kann, gibt die Funktion 0 (false) zurück, und die Ausgabevariablen werden leer gemacht. Die Funktion gibt 1 (true) zurück, wenn das System eine Textcursor-Position zurückgegeben hat, was aber nicht unbedingt bedeutet, dass ein Textcursor sichtbar ist.

Bemerkungen

Wenn bestimmte Informationen nicht benötigen werden, können die entsprechenden Ausgabevariablen weggelassen werden.

Beachten Sie, dass einige Fenster (z. B. bestimmte Versionen von MS Word) immer die gleichen Positionsdaten melden, wenn Sie den Textcursor verschieben.

Siehe auch

CoordMode, A_CoordModeCaret

Beispiele

Ermöglicht dem Benutzer, den Textcursor herumzubewegen, um seine aktuellen Positionsdaten in einem selbstaktualisierenden Tooltip zu sehen.

SetTimer TextcursorAbfragen, 100
TextcursorAbfragen() {
    if CaretGetPos(&x, &y)
        ToolTip "X" x " Y" y, x, y - 20
    else
        ToolTip "Kein Textcursor vorhanden"
}
