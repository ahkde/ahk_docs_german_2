DriveGetList

Gibt eine Reihe von Buchstaben zurück, die jeweils einen Laufwerksbuchstaben im System repräsentieren.

Liste := DriveGetList(Typ)

Parameter

Typ

Typ: Zeichenkette

Lässt man diesen Parameter weg, werden alle Laufwerkstypen abgerufen. Ansonsten kann eines der folgenden Wörter angegeben werden, um nur einen bestimmten Laufwerkstyp abzurufen: CDROM, REMOVABLE, FIXED, NETWORK, RAMDISK, UNKNOWN.

Rückgabewert

Typ: Zeichenkette

Diese Funktion gibt je nach angegebenem Typ die Laufwerksbuchstaben im System zurück. Zum Beispiel: ACDEZ.
