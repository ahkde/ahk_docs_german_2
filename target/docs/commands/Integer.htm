Integer

Konvertiert eine numerische Zeichenkette oder einen Floating-Point-Wert in einen Integer.

IntWert := Integer(Wert)

Rückgabewert

Typ: Integer

Diese Funktion gibt das Ergebnis der Konvertierung von Wert in einen reinen Integer (mit dem Typnamen "Integer"), oder Wert selbst, wenn es bereits der richtige Typ ist, zurück.

Bemerkungen

Jegliche Nachkommastellen von Wert gehen verloren, was vergleichbar ist mit Wert < 0 ? Ceil(Wert) : Floor(Wert).

Wenn der Wert nicht konvertiert werden kann, wird ein TypeError ausgelöst.

Um festzustellen, ob ein Wert in einen Integer umgewandelt werden kann, nutzen Sie die IsNumber-Funktion.

Integer ist eigentlich eine Klasse, kann aber als Funktion aufgerufen werden. Mit Wert is Integer kann geprüft werden, ob ein Wert ein reiner Integer ist.
