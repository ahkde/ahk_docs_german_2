EnvSet

Schreibt einen Wert in eine Umgebungsvariable.

EnvSet UmgebungVar , Wert

Parameter

UmgebungVar

Typ: Zeichenkette

Name der Umgebungsvariable, z. B. "COMSPEC" oder "PATH".

Wert

Typ: Zeichenkette

Ein Wert, der in die Umgebungsvariable gespeichert werden soll. Lässt man diesen Parameter weg, wird die Umgebungsvariable gelöscht.

Fehlerbehandlung

Bei Misserfolg wird ein OSError ausgelöst.

Bemerkungen

Das Betriebssystem limitiert jede Umgebungsvariable auf 32 KB Text.

Eine Umgebungsvariable, die mit dieser Funktion erstellt oder geändert wurde, ist nur für Programme zugänglich, die das Skript via Run oder RunWait gestartet hat. Weitere Informationen finden Sie unter Umgebungsvariablen.

Diese Funktion existiert, weil normale Skriptvariablen nicht in die Systemumgebung gespeichert werden. Das liegt zum einen daran, dass die Leistung darunter leiden würde und zum anderen daran, dass das Betriebssystem die Größe jeder Umgebungsvariable auf 32 KB begrenzt.

Siehe auch

EnvGet, Umgebungsvariablen, Run, RunWait

Beispiele

Schreibt etwas Text in die AutGUI-Umgebungsvariable.

EnvSet "AutGUI", "Beliebiger Text für die Variable."
