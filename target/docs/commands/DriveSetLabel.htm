DriveSetLabel

Ändert die Datenträgerbezeichnung eines bestimmten Laufwerks.

DriveSetLabel Laufwerk , NeueBezeichnung

Parameter

Laufwerk

Typ: Zeichenkette

Der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und einem optionalen umgekehrten Schrägstrich (UNC-Netzwerkpfade und zugeordnete Laufwerke wären auch möglich).

NeueBezeichnung

Typ: Zeichenkette

Eine neue Bezeichnung, die gesetzt werden soll. Lässt man diesen Parameter weg, bekommt das Laufwerk eine leere Bezeichnung zugewiesen.

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst.

Siehe auch

DriveGetLabel, Drive-Funktionen

Beispiele

Ändert die Datenträgerbezeichnung des C-Laufwerks.

DriveSetLabel "C:", "Seagate200"
