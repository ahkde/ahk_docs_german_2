HasProp

Gibt eine Nummer ungleich 0 zurück, wenn ein bestimmter Wert eine Eigenschaft mit einem bestimmten Namen hat.

HasProp := HasProp(Wert, Name)

Parameter

Wert

Typ: Any

Ein Wert beliebigen Typs, außer vom Typ ComObject.

Name

Typ: Zeichenkette

Der zu prüfende Eigenschaftsname.

Rückgabewert

Typ: Integer (boolescher Wert)

Diese Funktion gibt True zurück, wenn der Wert eine Eigenschaft mit diesem Namen hat, ansonsten False.

Bemerkungen

Diese Funktion prüft nicht, ob eine __Get- oder __Set-Metafunktion vorhanden ist. Wenn vorhanden, gibt es keine Möglichkeit, den exakten Satz von Eigenschaften zu erkennen, den es eventuell implementiert.

Diese Funktion unterstützt primitive Werte.

Siehe auch

Objekte, HasBase, HasMethod

Beispiele

Veranschaulicht die Verwendung dieser Funktion.

MsgBox HasProp({}, "x") ; 0
MsgBox HasProp({x:1}, "x") ; 1
MsgBox HasProp(0, "Base") ; 1
