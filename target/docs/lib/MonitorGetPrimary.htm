MonitorGetPrimary

Gibt die Nummer des primären Bildschirms zurück.

Primär := MonitorGetPrimary()

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Typ: Integer

Diese Funktion gibt die Nummer des primären Bildschirms zurück. In einem System mit nur einem einzigen Bildschirm wird das immer 1 sein.
