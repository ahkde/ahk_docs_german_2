Float

Wandelt eine numerische Zeichenkette oder einen Integer in eine Floating-Point-Zahl um.

FltWert := Float(Wert)

Rückgabewert

Typ: Float

Diese Funktion gibt das Ergebnis der Umwandlung von Wert in eine reine Floating-Point-Zahl (vom Typ "Float") zurück, oder Wert selbst, wenn es bereits den richtigen Typ hat.

Bemerkungen

Wenn der Wert nicht umgewandelt werden kann, wird ein TypeError ausgelöst.

Mit der IsNumber-Funktion kann ermittelt werden, ob ein Wert in eine Floating-Point-Zahl umgewandelt werden kann.

Float ist eigentlich eine Klasse, kann aber als Funktion aufgerufen werden. Mit Wert is Float kann geprüft werden, ob ein Wert eine reine Floating-Point-Zahl ist.
