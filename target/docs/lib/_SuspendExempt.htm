#SuspendExempt

Schützt nachfolgende Hotkeys und Hotstrings vor einer Suspension.

#SuspendExempt Schützen

Parameter

Schützen

True oder 1, um den Schutz für nachfolgende Hotkeys zu aktivieren, oder False oder 0, um den Schutz zu deaktivieren. Lässt man diesen Parameter weg, wird standardmäßig True verwendet.

Bemerkungen

Hotkeys und Hotstrings können via Suspend-Funktion, Tray-Symbol oder Hauptfenster suspendiert werden.

Wenn diese Direktive im Skript nicht angegeben ist, werden alle Hotkeys oder Hotstrings bei einer Suspension deaktiviert, auch solche, die die Suspend-Funktion aufrufen.

Diese Direktive hat keinen Einfluss auf die Funktionen Hotkey oder Hotstring. Verwenden Sie hierfür stattdessen die Hotkey-Option S oder Hotstring-Option S.

#SuspendExempt kann wie alle anderen Direktiven nicht bedingt ausgeführt werden.

Siehe auch

Suspend, Hotkeys, Hotstrings

Beispiele

Der erste Hotkey in diesem Beispiel schaltet die Suspension um. Um zu verhindern, dass dieser Hotkey nach dem Einschalten der Suspension suspendiert wird und somit nicht mehr ausgeschaltet werden kann, muss er davor geschützt werden.

#SuspendExempt  ; Schützt den folgenden Hotkey vor Suspend.
#Esc::Suspend -1
#SuspendExempt False  ; Deaktiviert den Schutz für alle nachfolgenden Hotkeys/Hotstrings.
^1::MsgBox "Dieser Hotkey kann suspendiert werden."
