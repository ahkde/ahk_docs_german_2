DirDelete

Löscht einen Ordner.

DirDelete VerzName , Rekursiv

Parameter

VerzName

Typ: Zeichenkette

Name des Verzeichnisses, das gelöscht werden soll. Wenn kein absoluter Pfad angegeben ist, wird das Verzeichnis im A_WorkingDir-Verzeichnis vermutet.

Rekursiv

Typ: Boolesch

Dieser Parameter gibt an, ob Unterverzeichnisse rekursiv durchwandert werden. Wenn weggelassen, wird standardmäßig False verwendet (kein Durchwandern). Andernfalls geben Sie einen der folgenden Werte an:

0 oder False: Dateien und Unterverzeichnisse werden nicht aus VerzName entfernt. Wenn VerzName in diesem Fall nicht leer ist, wird nichts unternommen und eine Ausnahme ausgelöst.

1 oder True: Entfernt alle Dateien und Unterverzeichnisse (wie der Windows-Befehl "rmdir /S").

Fehlerbehandlung

Es wird eine Ausnahme ausgelöst, wenn ein Fehler auftritt.

Siehe auch

DirCreate, FileDelete

Beispiele

Entfernt das Verzeichnis, aber nur wenn es leer ist.

DirDelete "C:\Download-Temp"

Entfernt das Verzeichnis, einschließlich dessen Dateien und Unterverzeichnisse.

DirDelete "C:\Download-Temp", 1
