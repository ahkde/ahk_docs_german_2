DirDelete

Löscht einen Ordner.

DirDelete VerzName , Rekursiv

Parameter

VerzName

Typ: Zeichenkette

Name des Verzeichnisses, das gelöscht werden soll. Wenn kein absoluter Pfad angegeben ist, wird das Verzeichnis im A_WorkingDir-Verzeichnis vermutet.

Rekursiv

Typ: Boolesch

Wenn weggelassen, wird standardmäßig false verwendet.

Wenn false, werden die in VerzName enthaltenen Dateien und Unterverzeichnisse nicht entfernt. Wenn VerzName in diesem Fall nicht leer ist, wird nichts unternommen und eine Ausnahme ausgelöst.

Wenn true, werden alle Dateien und Unterverzeichnisse entfernt (wie der Windows-Befehl "rmdir /S").

Fehlerbehandlung

Es wird eine Ausnahme ausgelöst, wenn ein Fehler auftritt.

Siehe auch

DirCreate, FileDelete

Beispiele

Entfernt das Verzeichnis, aber nur wenn es leer ist.

DirDelete "C:\Download-Temp"

Entfernt das Verzeichnis, einschließlich dessen Dateien und Unterverzeichnisse.

DirDelete "C:\Download-Temp", true
