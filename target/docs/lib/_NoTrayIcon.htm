#NoTrayIcon

Deaktiviert die Anzeige des Tray-Symbols.

#NoTrayIcon

Diese Direktive irgendwo im Skript verhindert, dass das Tray-Symbol beim Start des Skripts angezeigt wird (auch dann, wenn das Skript kompiliert ist).

Wenn Sie diese Direktive in einem Skript nutzen, das Hotkeys hat, empfiehlt es sich, einen Hotkey an die ExitApp-Funktion zu binden. Andernfalls gäbe es keine einfache Möglichkeit, das Programm zu beenden (ohne dabei den Computer neu zu starten oder den Prozess zu beenden). Zum Beispiel: #x::ExitApp.

Um das Tray-Symbol jederzeit beim Ausführen des Skripts ein- oder auszublenden, weisen Sie A_IconHidden einen wahren oder falschen Wert zu. Die Verwendung von A_IconHidden := true ganz oben im Skript hat den Nachteil, dass das Tray-Symbol beim Starten des Skripts kurz sichtbar sein wird. Um das zu verhindern, können Sie stattdessen #NoTrayIcon verwenden.

Die interne Variable A_IconHidden enthält 1, wenn das Tray-Symbol momentan ausgeblendet ist, andernfalls 0; außerdem kann sie auf einen Wert gesetzt werden, um das Symbol ein- oder auszublenden.

#NoTrayIcon kann wie alle anderen Direktiven nicht bedingt ausgeführt werden.
