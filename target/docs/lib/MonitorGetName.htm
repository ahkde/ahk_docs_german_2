MonitorGetName

Gibt den systemdefinierten Namen eines bestimmten Bildschirms zurück.

Name := MonitorGetName(N)

Parameter

N

Typ: Integer

Wenn weggelassen, wird der primäre Bildschirm verwendet. Andernfalls geben Sie eine Bildschirmnummer an, zwischen 1 und dem Rückgabewert von MonitorGetCount.

Rückgabewert

Typ: Zeichenkette

Diese Funktion gibt den systemdefinierten Namen des angegebenen Bildschirms zurück.

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst.
