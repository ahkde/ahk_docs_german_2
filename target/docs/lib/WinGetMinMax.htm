WinGetMinMax

Gibt den Zustand zurück, ob ein bestimmtes Fenster maximiert oder minimiert ist.

MinMax := WinGetMinMax(FensterTitel, FensterText, AusnahmeTitel, AusnahmeText)

Parameter

FensterTitel, FensterText, AusnahmeTitel, AusnahmeText

Typ: Zeichenkette, Integer oder Objekt

Wenn jeder dieser Parameter leer ist oder weggelassen wird, wird das Zuletzt Gefundene Fenster verwendet. Andernfalls geben Sie für FensterTitel einen Fenstertitel oder andere Kriterien zur Identifizierung des Zielfensters und/oder für FensterText eine Teilzeichenkette aus einem Textelement des Zielfensters an (Textelemente können mit dem internen Tool "Window Spy" aufgedeckt werden). Versteckte Textelemente werden erkannt, wenn DetectHiddenText eingeschaltet wurde. Fenster, deren Titel oder Texte AusnahmeTitel bzw. AusnahmeText enthalten, werden ignoriert.

Rückgabewert

Typ: Integer

Diese Funktion gibt den aktuellen Zustand des angegebenen Fensters zurück:

  • -1: Das Fenster ist minimiert (WinRestore kann es entminimieren).
  • 1: Das Fenster ist maximiert (WinRestore kann es entmaximieren).
  • 0: Das Fenster ist weder minimiert noch maximiert.

Fehlerbehandlung

Es wird ein TargetError ausgelöst, wenn das Fenster nicht gefunden werden konnte.

Bemerkungen

Mit MouseGetPos kann die ID des Fensters unter dem Mauszeiger abgerufen werden.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.
