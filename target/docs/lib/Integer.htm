Integer

Wandelt eine numerische Zeichenkette oder einen Floating-Point-Wert in einen Integer um.

IntWert := Integer(Wert)

Rückgabewert

Typ: Integer

Diese Funktion gibt das Ergebnis der Umwandlung von Wert in einen reinen Integer (vom Typ "Integer") zurück, oder Wert selbst, wenn es bereits den richtigen Typ hat.

Bemerkungen

Alle Dezimalstellen von Wert gehen verloren, äquivalent zu Wert < 0 ? Ceil(Wert) : Floor(Wert).

Wenn der Wert nicht umgewandelt werden kann, wird ein TypeError ausgelöst.

Mit der IsNumber-Funktion kann ermittelt werden, ob ein Wert in einen Integer umgewandelt werden kann.

Integer ist eigentlich eine Klasse, kann aber als Funktion aufgerufen werden. Mit Wert is Integer kann geprüft werden, ob ein Wert ein reiner Integer ist.
