WinSetTitle

Ändert den Titel eines bestimmten Fensters.

WinSetTitle NeuerTitel , FensterTitel, FensterText, AusnahmeTitel, AusnahmeText

Parameter

NeuerTitel

Typ: Zeichenkette

Der neue Titel des Fensters. Wenn nur dieser Parameter vorhanden ist, wird das Zuletzt Gefundene Fenster verwendet.

FensterTitel, FensterText, AusnahmeTitel, AusnahmeText

Typ: Zeichenkette, Integer oder Objekt

Wenn jeder dieser Parameter ungesetzt ist oder weggelassen wird, wird das Zuletzt Gefundene Fenster verwendet. Andernfalls geben Sie für FensterTitel ein Fenstertitel oder andere Kriterien zur Identifizierung des Zielfensters und/oder für FensterText eine Teilzeichenkette aus einem Textelement des Zielfensters an (Textelemente können mit dem internen Tool "Window Spy" aufgedeckt werden). Versteckte Textelemente werden erkannt, wenn DetectHiddenText eingeschaltet wurde. Fenster, deren Titel oder Texte AusnahmeTitel bzw. AusnahmeText enthalten, werden ignoriert.

Fehlerbehandlung

Es wird ein TargetError ausgelöst, wenn das Fenster nicht gefunden werden konnte.

Es wird ein OSError ausgelöst, wenn die Änderung nicht angewendet werden konnte.

Bemerkungen

Der Fenstertitel kann sich jederzeit wieder ändern, weil das dazugehörige Programm diesen Titel auch ändern könnte.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

WinMove, WinGetTitle, WinGetText, ControlGetText, WinGetPos, Win-Funktionen

Beispiele

Ändert den Titel von Notepad. Dieses Beispiel kann unter Windows 11 oder höher fehlschlagen, da es die klassische Version von Notepad erfordert.

WinSetTitle("Ein neuer Titel", "Unbenannt - Editor")

Öffnet Notepad, wartet, bis es aktiv ist, und ändert seinen Titel. Dieses Beispiel kann unter Windows 11 oder höher fehlschlagen, da es die klassische Version von Notepad erfordert.

Run "notepad.exe"
WinWaitActive "Unbenannt - Editor"
WinSetTitle "Ein neuer Titel" ; Verwendet das von WinWaitActive gefundene Fenster.

Öffnet das Hauptfenster, wartet, bis es aktiv ist, und ändert dessen Titel.
