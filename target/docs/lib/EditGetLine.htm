EditGetLine

Gibt den Text einer bestimmten Zeile in einem Edit-Steuerelement zurück.

Zeile := EditGetLine(N, Steuerelement , FensterTitel, FensterText, IgnoriereTitel, IgnoriereText)

Parameter

N

Typ: Integer

Die Zeilennummer. Die erste Zeile ist 1.

Steuerelement

Typ: Zeichenkette, Integer oder Objekt

ClassNN-Bezeichnung, Text oder HWND-Nummer des Steuerelements, oder ein Objekt mit einer Hwnd-Eigenschaft. Einzelheiten finden Sie unter Der Steuerelement-Parameter.

FensterTitel

Typ: Zeichenkette, Integer oder Objekt

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter FensterTitel-Parameter.

FensterText

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

IgnoriereTitel

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Titeln enthalten, werden ignoriert.

IgnoriereText

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Textelementen enthalten, werden ignoriert.

Rückgabewert

Typ: Zeichenkette

Diese Funktion gibt den Text der Zeile N eines Edit-Steuerelements zurück. Je nach Typ des Steuerelements endet die Zeichenkette eventuell mit einem CR-Zeichen (`r) oder einem CR-LF-Paar (`r`n).

Fehlerbehandlung

Es wird ein TargetError ausgelöst, wenn das Fenster oder Steuerelement nicht gefunden werden konnte.

Es wird ein ValueError ausgelöst, wenn N außerhalb des Bereichs liegt oder anderweitig ungültig ist.

Es wird ein OSError ausgelöst, wenn eine Meldung nicht an das Steuerelement gesendet werden konnte.

Bemerkungen

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Beispiele

Ruft die erste Zeile des Edit-Steuerelements im Texteditor ab.

Zeile1 := EditGetLine(1, "Edit1", "ahk_class Notepad")
