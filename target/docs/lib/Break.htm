Break

Beendet (unterbricht) eine Schleifenanweisung beliebiger Art.

Break LoopLabel

Parameter

LoopLabel

LoopLabel kennzeichnet, für welche Schleife diese Anweisung gelten soll; dieser Parameter kann entweder der Name eines Labels oder die Nummer einer Verschachtelungsebene sein. Wenn dieser Parameter weggelassen wird oder 1 ist, gilt diese Anweisung für die innerste umschlossene Schleife. Wenn Sie vorhaben, den Namen eines Labels anzugeben, müssen Sie darauf achten, dass dieses Label direkt auf eine Schleifenanweisung verweist.

LoopLabel muss ein konstanter Wert sein - Variablen und Ausdrücke werden nicht unterstützt, außer eine direkt geschriebene Zahl oder eine in Anführungszeichen gesetzte Zeichenkette, die mit runden Klammern umschlossen ist. Zum Beispiel: break("outer")

Bemerkungen

Break und Continue sind besser geeignet als Goto, weil sie in der Regel die Übersichtlichkeit und Wartbarkeit eines Skripts erhöhen.

Siehe auch

Continue, Loop, While-Schleife, For-Schleife, Blöcke, Label

Beispiele

Unterbricht die Schleife, wenn var größer als 25 ist.

Loop
{
    ; ...
    if (var > 25)
        break
    ; ...
    if (var <= 5)
        continue
}

Unterbricht von einer inneren Schleife heraus die äußere Schleife.

outer:
Loop 3
{
    x := A_Index
    Loop 3
    {
        if (x*A_Index = 6)
            break outer  ; Äquivalent zu break 2 oder goto break_outer.
        MsgBox x "," A_Index
    }
}
break_outer: ; Für goto.
