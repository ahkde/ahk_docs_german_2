FileRecycle

Verschiebt eine Datei oder ein Verzeichnis in den Papierkorb, falls möglich, oder löscht sie endgültig.

FileRecycle DateiMuster

Parameter

DateiMuster

Typ: Zeichenkette

Name einer Datei oder ein Platzhaltermuster wie C:\Temp\*.tmp. Wenn kein absoluter Pfad angegeben ist, wird DateiMuster im A_WorkingDir-Verzeichnis vermutet.

Um einen kompletten Ordner zu recyceln, geben Sie seinen Namen ohne umgekehrten Schrägstrich am Ende an.

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst.

Bemerkungen

Mit SHFileOperation wird die eigentliche Arbeit getan. Diese Funktion kann eine Datei endgültig löschen, wenn sie nicht in den Papierkorb passt; es erscheint eine Warnung, bevor das geschieht.

Die Datei kann ohne Vorwarnung endgültig gelöscht werden, wenn die Datei aus bestimmten Gründen nicht recycelt werden kann:

  • Die Datei befindet sich auf einem Wechseldatenträger.
  • Der Papierkorb wurde deaktiviert, z. B. via Registry-Wert NukeOnDelete.

Siehe auch

FileRecycleEmpty, FileDelete, FileCopy, FileMove

Beispiele

Sendet alle in einem Verzeichnis befindlichen TMP-Dateien in den Papierkorb, wenn möglich.

FileRecycle "C:\Temp-Dateien\*.tmp"
