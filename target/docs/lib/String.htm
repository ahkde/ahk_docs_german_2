String

Wandelt einen Wert in eine Zeichenkette um.

ZketteWert := String(Wert)

Rückgabewert

Typ: Zeichenkette

Diese Funktion gibt das Ergebnis der Umwandlung von Wert in eine Zeichenkette zurück, oder Wert selbst, wenn es eine Zeichenkette ist.

Wenn Wert eine Zahl ist, wird die reguläre Dezimalformatierung verwendet.

Wenn Wert ein Objekt ist, ist der Rückgabewert das Ergebnis des Aufrufs von Wert.ToString(). Wenn das Objekt keine solche Methode hat, wird ein MethodError ausgelöst. Diese Methode ist standardmäßig nicht implementiert und muss daher im Skript definiert werden.
