DriveGetStatusCD

Gibt den Medienstatus eines bestimmten CD- oder DVD-Laufwerks zurück.

CDStatus := DriveGetStatusCD(Laufwerk)

Parameter

Laufwerk

Typ: Zeichenkette

Der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt. Lässt man diesen Parameter weg, wird das Standard-CD/DVD-Laufwerk verwendet.

Rückgabewert

Typ: Zeichenkette

Diese Funktion gibt den Medienstatus des angegebenen CD- oder DVD-Laufwerks zurück:

Status Bedeutung
not ready Das Laufwerk ist nicht bereit für den Zugriff, z.B. aufgrund eines Schreibvorgangs. Bekannte Einschränkung: "not ready" tritt auch auf, wenn das Laufwerk eine DVD statt CD enthält.
open Das Laufwerk enthält keine CD oder ist geöffnet.
playing Das Laufwerk spielt gerade eine CD ab.
paused Die vorherige Audio- oder Videowiedergabe ist nun pausiert.
seeking Das Laufwerk sucht gerade.
stopped Das Laufwerk enthält eine CD, greift aber momentan nicht darauf zu.

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst.

Bemerkungen

Diese Funktion wird eventuell nicht funktionieren, wenn das angegebene Laufwerk ein Netzwerk- oder Nicht-CD/DVD-Laufwerk ist. Wenn es in solchen Fällen oder aus einem anderen Grund fehlschlägt, wird eine Ausnahme ausgelöst.

Wenn das Laufwerk vor Kurzem geschlossen wurde, kann es zu einer Verzögerung kommen, bevor die Funktion zu Ende ist.
