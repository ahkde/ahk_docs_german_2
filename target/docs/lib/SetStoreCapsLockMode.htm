SetStoreCapsLockMode

Bestimmt, ob der Zustand der Feststell-Taste nach einer Send-Aktion wiederhergestellt werden soll.

SetStoreCapsLockMode Modus

Parameter

Modus

Typ: Boolescher Wert

Geben Sie einen der folgenden Werte an:

1 oder True: Die Standardeinstellung für alle Skripte: Feststell wird auf ihren vorherigen Zustand zurückgesetzt, wenn Send diese Taste für die eigene Operation vorübergehend ändern musste.

0 oder False: Der Zustand von Feststell wird überhaupt nicht geändert. Das führt dazu, dass Send die Groß- und Kleinschreibung der Zeichen umkehren wird, wenn die Feststell-Taste während dieser Operation aktiv ist.

Rückgabewert

Typ: Boolescher Wert

Diese Funktion gibt die vorherige Einstellung zurück.

Bemerkungen

Beachten Sie, dass Feststell bei Send oder ControlSend nicht immer ausgeschaltet wird. Selbst wenn sie erfolgreich ausgeschaltet wurde, wird sie möglicherweise nicht wieder eingeschaltet, nachdem die Tasten gesendet wurden.

Diese Funktion wird selten verwendet, weil das Standardverhalten in den meisten Fällen vollkommen ausreicht.

Diese Einstellung wird im Blind-Modus und Text-Modus ignoriert; das heißt, dass der Status der Feststell-Taste in solchen Fällen nicht geändert wird.

Die interne Variable A_StoreCapsLockMode enthält die aktuelle Einstellung (1 oder 0).

Jeder neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitgesteuerte Subroutine) verwendet vorerst die Standardeinstellung der Funktion. Um diese zu ändern, nutzen Sie die Funktion während der Startphase des Skripts.
