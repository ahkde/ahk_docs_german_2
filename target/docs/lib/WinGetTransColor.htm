WinGetTransColor

Gibt die Farbe zurück, die in einem bestimmten Fenster als transparent markiert wurde.

TransFarbe := WinGetTransColor(FensterTitel, FensterText, AusnahmeTitel, AusnahmeText)

Parameter

FensterTitel, FensterText, AusnahmeTitel, AusnahmeText

Typ: Zeichenkette, Integer oder Objekt

Wenn jeder dieser Parameter leer ist oder weggelassen wird, wird das Zuletzt Gefundene Fenster verwendet. Andernfalls geben Sie für FensterTitel einen Fenstertitel oder andere Kriterien zur Identifizierung des Zielfensters und/oder für FensterText eine Teilzeichenkette aus einem Textelement des Zielfensters an (Textelemente können mit dem internen Tool "Window Spy" in Erfahrung gebracht werden). Versteckte Textelemente werden erkannt, wenn DetectHiddenText eingeschaltet wurde. Fenster, deren Titel oder Texte AusnahmeTitel bzw. AusnahmeText enthalten, werden ignoriert.

Rückgabewert

Typ: Zeichenkette

Diese Funktion gibt die 6-stellige RGB-Farbe zurück, die im angegebenen Fenster als transparent markiert wurde (siehe WinSetTransColor, wie eine transparente Farbe gesetzt werden kann). Der Rückgabewert ist eine leere Zeichenkette, wenn: 1) keine passenden Fenster gefunden werden; 2) das Fenster keine transparente Farbe hat; oder 3) andere Bedingungen auftreten (durch das Verhalten des Betriebssystems), z.B. wenn das transparent gemachte Fenster minimiert, wiederhergestellt und/oder skaliert wurde.

Fehlerbehandlung

Es wird ein TargetError ausgelöst, wenn das Fenster nicht gefunden werden konnte.

Bemerkungen

Mit MouseGetPos kann die ID des Fensters unter dem Mauszeiger abgerufen werden.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Siehe auch

WinSetTransColor, WinGetTransparent, Win-Funktionen, Control-Funktionen

Beispiele

Ruft die transparente Farbe eines Fensters unter dem Mauszeiger ab.

MouseGetPos ,, &MausFenster
TransFarbe := WinGetTransColor(MausFenster)
