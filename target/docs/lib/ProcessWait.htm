ProcessWait

Wartet, bis ein bestimmter Prozess existiert.

PID := ProcessWait(PIDOderName , Zeitlimit)

Parameter

PIDOderName

Typ: Integer oder Zeichenkette

Geben Sie entweder eine Nummer (die PID) oder einen Prozessnamen an:

PID (Prozess-ID): Eine Nummer zur eindeutigen Identifizierung eines Prozesses (die Gültigkeit dieser Nummer ist an die Lebensdauer des Prozesses gebunden). Mit der Run-Funktion kann die PID eines neu gestarteten Prozesses ermittelt werden. Im Vergleich dazu kann mit WinGetPID die PID eines Fensters ermittelt werden. Mit ProcessExist kann auch eine PID abgerufen werden.

Name: Der Name eines Prozesses (ohne Pfad), den üblicherweise auch seine ausführbare Datei hat, z.B. notepad.exe oder winword.exe. Da ein Name mit mehreren laufenden Prozessen übereinstimmen kann, wird nur der erste Prozess berücksichtigt. Der Name ist nicht Groß-/Kleinschreibung-sensitiv.

Zeitlimit

Typ: Integer oder Float

Wenn weggelassen, wird die Funktion unendlich lang warten. Andernfalls geben Sie die maximale Wartezeit in Sekunden an (kann einen Dezimalpunkt enthalten).

Rückgabewert

Typ: Integer

Wenn der angegebene Prozess gefunden wurde, gibt die Funktion die Prozess-ID (PID) des Prozesses zurück. Hat die Funktion das Zeitlimit überschritten, wird 0 zurückgegeben.

Bemerkungen

Prozesse werden alle 100 Millisekunden überprüft; sobald die Bedingung erfüllt ist, hört die Funktion auf zu warten. Das heißt, dass die Funktion nicht auf das Zeitlimit wartet, sondern dass sie sofort ihren Wert zurückgibt und die Skriptausführung fortsetzt. Während die Funktion im Wartezustand ist, können neue Threads via Hotkey, benutzerdefinierte Menüpunkte oder Timer gestartet werden.

Beispiele

Wartet maximal 5.5 Sekunden, bis Notepad erscheint. Wenn Notepad innerhalb dieser Zeit erscheint, wird seine Priorität auf "Niedrig" und die Priorität des Skripts auf "Hoch" gesetzt. Danach wird Notepad geschlossen und ein Mitteilungsfenster angezeigt, wenn es nicht innerhalb von 5 Sekunden geschlossen wird.

NeuePID := ProcessWait("notepad.exe", 5.5)
if not NeuePID
{
    MsgBox "Der angegebene Prozess konnte innerhalb der 5.5 Sekunden nicht gefunden werden."
    return
}
; Andernfalls:
MsgBox "Es wurde ein passender Prozess gefunden (Prozess-ID ist " NeuePID ")."
ProcessSetPriority "Low", NeuePID
ProcessSetPriority "High"  ; Das Skript sich selbst auf hohe Priorität setzen lassen.
WinClose "Unbenannt - Editor"
PID := ProcessWaitClose(NeuePID, 5)
if PID ; Die PID existiert noch.
    MsgBox "Der Prozess konnte innerhalb der 5 Sekunden nicht geschlossen werden."
