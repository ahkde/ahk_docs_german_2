DriveSetLabel

Ändert die Volumebezeichnung eines bestimmten Laufwerks.

DriveSetLabel Laufwerk , NeueBezeichnung

Parameter

Laufwerk

Typ: Zeichenkette

Der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und optional einem umgekehrten Schrägstrich (kann auch mit UNC-Pfaden und zugeordneten Laufwerken funktionieren).

NeueBezeichnung

Typ: Zeichenkette

Wenn weggelassen, bekommt das Laufwerk eine leere Bezeichnung zugewiesen. Andernfalls geben Sie die neue Bezeichnung an, die gesetzt werden soll.

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst.

Beispiele

Ändert die Volumebezeichnung des C-Laufwerks.

DriveSetLabel "C:", "Seagate200"
