DirCreate

Erstellt einen Ordner.

DirCreate VerzName

Parameter

VerzName

Typ: Zeichenkette

Name des Verzeichnisses, das erstellt werden soll. Wenn kein absoluter Pfad angegeben ist, wird das Verzeichnis im A_WorkingDir-Verzeichnis erstellt.

Fehlerbehandlung

Es wird ein OSError ausgelöst, wenn ein Fehler auftritt.

A_LastError wird auf das Ergebnis der Systemfunktion GetLastError() gesetzt.

Bemerkungen

Diese Funktion erstellt auch alle übergeordneten Verzeichnisse, die in VerzName vorkommen, falls sie noch nicht vorhanden sind.

Siehe auch

DirDelete

Beispiele

Erstellt ein neues Verzeichnis, einschließlich der Elternverzeichnisse, falls erforderlich.

DirCreate "C:\Test1\Meine Bilder\Ordner2"
