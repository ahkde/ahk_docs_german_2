DriveGetCapacity

Gibt die Gesamtkapazität eines Laufwerks zurück, das einen bestimmten Pfad enthält (in Megabyte).

Kapazität := DriveGetCapacity(Pfad)

Parameter

Pfad

Typ: Zeichenkette

Ein beliebiger Pfad, der im Laufwerk enthalten ist (kann auch mit UNC-Pfaden und zugeordneten Laufwerken funktionieren).

Rückgabewert

Typ: Integer

Diese Funktion gibt die Gesamtkapazität des Laufwerks zurück, das Pfad enthält (in Megabyte).

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst.

Bemerkungen

In der Regel kann Pfad ein beliebiger Pfad sein. Da NTFS "mounted volumes" (bereitgestellte Volumes) und "directory junctions" (Verzeichnisverbindungen) unterstützt, können verschiedene Pfade mit demselben Laufwerksbuchstaben unterschiedliche Kapazitätsmengen erzeugen.
