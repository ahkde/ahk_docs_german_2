ListHotkeys

Zeigt an, welche Hotkeys vom aktuellen Skript verwendet werden, ob deren Subroutinen gerade laufen und ob sie den Tastatur- oder Maus-Hook verwenden.

ListHotkeys

Diese Funktion entspricht der Auswahl des Menüpunkts "View->Hotkeys" im Hauptfenster.

Wenn ein Hotkey per Hotkey-Funktion deaktiviert wurde, wird er als OFF oder PART aufgelistet (PART bedeutet, dass einige Varianten des Hotkeys noch aktiv sind).

Wenn einer der Hotkey-Varianten einen #InputLevel ungleich null hat, wird die Stufe (oder die Minimal- und Maximalstufe) angezeigt.

Wenn Subroutinen eines Hotkeys gerade ausgeführt werden, wird die Gesamtzahl der Threads für diesen Hotkey angezeigt.

Außerdem wird der Typ des Hotkeys angezeigt, der einer der folgenden sein kann:

  • reg: Der Hotkey wurde mit der Systemfunktion RegisterHotKey() implementiert.
  • reg(no): Wie oben, außer dass dieser Hotkey inaktiv ist (weil er nicht unterstützt wird, deaktiviert oder suspendiert ist).
  • k-hook: Der Hotkey wurde via Tastatur-Hook implementiert.
  • m-hook: Der Hotkey wurde via Maus-Hook implementiert.
  • 2-hooks: Der Hotkey benötigt beide der oben genannten Hooks.
  • joypoll: Der Hotkey wurde durch regelmäßiges Abfragen des Controllers implementiert.
