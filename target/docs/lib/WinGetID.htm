WinGetID

Gibt die eindeutige ID-Nummer eines bestimmten Fensters zurück.

ID := WinGetID(FensterTitel, FensterText, IgnoriereTitel, IgnoriereText)

Parameter

FensterTitel

Typ: Zeichenkette, Integer oder Objekt

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter FensterTitel-Parameter.

FensterText

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

IgnoriereTitel

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Titeln enthalten, werden ignoriert.

IgnoriereText

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Textelementen enthalten, werden ignoriert.

Rückgabewert

Typ: Integer

Diese Funktion gibt die HWND-Nummer (eindeutige ID) des angegebenen Fensters zurück.

Fehlerbehandlung

Es wird ein TargetError ausgelöst, wenn das Fenster nicht gefunden werden konnte.

Bemerkungen

Diese Funktion ist das Äquivalent zu WinExist.

Die ID-Nummer eines Fensters bleibt nur zu dessen Lebzeiten gültig. Mit anderen Worten: Wenn man eine Anwendung neu starten würde, bekämen all ihre Fenster neue ID-Nummern.

Die ID des Fensters, das sich unter dem Mauszeiger befindet, kann mit MouseGetPos abgerufen werden. Um die HWND-Nummer eines Steuerelements abzurufen (z. B. für PostMessage, SendMessage oder DllCall), können Sie ControlGetHwnd oder MouseGetPos nutzen.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Beispiele

Maximiert das aktive Fenster und meldet dessen eindeutige ID.

AktiveID := WinGetID("A")
WinMaximize AktiveID
MsgBox "Die ID des aktiven Fensters ist " AktiveID
