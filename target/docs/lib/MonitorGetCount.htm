MonitorGetCount

Gibt die Gesamtzahl der Bildschirme zurück.

Anzahl := MonitorGetCount()

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Typ: Integer

Diese Funktion gibt die Gesamtzahl der Bildschirme zurück.

Bemerkungen

Im Gegensatz zur Systemeigenschaft SM_CMONITORS, die via SysGet(80) abgerufen werden kann, erfasst diese Funktion alle Bildschirme, auch solche, die nicht zum Desktop gehören.
