SoundSetMute

Ändert die Stummschalteinstellung eines Audiogeräts.

SoundSetMute NeueEinstellung , Komponente, Gerät

Parameter

NeueEinstellung

Typ: Zahl oder Zeichenkette

Eine beliebige positive Zahl, um die Einstellung einzuschalten (stummzuschalten), oder eine 0, um sie auszuschalten (Stummschaltung aufzuheben). Wenn die Zahl jedoch mit einem Plus- oder Minuszeichen beginnt, wird die Einstellung umgeschaltet (jeweils auf den entgegengesetzten Zustand).

Komponente

Typ: Zeichenkette oder Integer

Anzeigename und/oder Indexnummer der Komponente. Zum Beispiel 1, "Line in" oder "Line in:2".

Wenn dieser Parameter weggelassen wird oder leer ist, wird die Master-Stummschalteinstellung geändert.

Weitere Informationen finden Sie unter Komponente (Sound-Funktionen).

Gerät

Typ: Zeichenkette oder Integer

Anzeigename und/oder Indexnummer des Geräts. Zum Beispiel 1, "Lautsprecher", "Lautsprecher:2" oder "Lautsprecher (Beispiel HD Audio)".

Wenn weggelassen, wird standardmäßig das als Standard festgelegte Wiedergabegerät des Systems verwendet (was nicht unbedingt Gerät 1 ist).

Weitere Informationen finden Sie unter Gerät (Sound-Funktionen).

Fehlerbehandlung

Bei Misserfolg kann eine der folgenden Ausnahmen ausgelöst werden:

Error-KlasseMessage
TargetErrorDevice not found (Gerät nicht gefunden)
Component not found (Komponente nicht gefunden)
Component doesn't support this control type (Komponente unterstützt diesen Regler nicht)
OSErrorVom System generiert.

Bemerkungen

Zum Umschalten der Master-Stummschalteinstellung des als Standard festgelegten Wiedergabegeräts kann alternativ auch ein Skript verwendet werden, das Tastendrücke sendet. Zum Beispiel:

Send "{Volume_Mute}"  ; Master-Lautstärke stummschalten bzw. Stummschaltung aufheben.

Um den Leistungsumfang der installierten Audiogeräte zu ermitteln - z.B. die Namen und verfügbaren Komponenten - führen Sie das Soundkartenanalyse-Skript aus.

Mit SoundGetMute kann die aktuelle Stummschalteinstellung abgerufen werden.

Beispiele

Schaltet die Master-Lautstärke stumm.

SoundSetMute true

Schaltet die Master-Stummschaltung um (jeweils auf den entgegengesetzten Zustand).

SoundSetMute -1

Schaltet Line-In stumm.

SoundSetMute 1, "Line In"

Schaltet Mikrofonaufnahme stumm.

SoundSetMute 1, , "Mikrofon"
