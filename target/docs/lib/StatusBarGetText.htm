StatusBarGetText

Ermittelt den Text einer Standard-Statusleiste.

Text := StatusBarGetText(SegmentNummer, FensterTitel, FensterText, IgnoriereTitel, IgnoriereText)

Parameter

SegmentNummer

Typ: Integer

Die Nummer des Leistenbereichs, dessen Text abgerufen werden soll. Wenn dieser Parameter leer ist, wird standardmäßig 1 verwendet (der erste Bereich enthält üblicherweise den interessanten Text).

FensterTitel

Typ: Zeichenkette, Integer oder Objekt

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter FensterTitel-Parameter.

FensterText

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

IgnoriereTitel

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Titeln enthalten, werden ignoriert.

IgnoriereText

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Textelementen enthalten, werden ignoriert.

Rückgabewert

Typ: Zeichenkette

Diese Funktion gibt einen Bereichstext der Statusleiste zurück.

Fehlerbehandlung

Es wird in folgenden Fällen eine Ausnahme ausgelöst:

  • TargetError: Das Zielfenster konnte nicht gefunden werden oder enthält keine standardmäßige Statusleiste.
  • OSError: Es gab ein Problem beim Senden der SB_GETPARTS-Meldung, oder es erfolgte keine Antwort innerhalb von 2000 ms.
  • OSError: Es konnte kein Speicher innerhalb des Prozesses, dem die Statusleiste gehört, reserviert werden.

Bemerkungen

Diese Funktion versucht, die erste Standard-Statusleiste eines Fensters zu lesen (typisches Steuerelement von Microsoft: msctls_statusbar32). Einige Programme verwenden ihre eigenen Statusleisten oder spezielle Versionen des Microsoft-Steuerelements - in diesem Fall kann der Text nicht abgerufen werden.

Anstatt diese Funktion in einer Schleife zu verwenden, ist es besser, StatusBarWait zu verwenden, weil es Optimierungen beinhaltet, die einen Overhead durch mehrfaches Aufrufen von StatusBarGetText verhindern.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Beispiele

Ermittelt und analysiert den Text im ersten Segment einer Statusleiste.

AbgerufenerText := StatusBarGetText(1, "Suchergebnisse")
if InStr(AbgerufenerText, "Gefunden")
    MsgBox "Suchergebnisse wurden gefunden."
