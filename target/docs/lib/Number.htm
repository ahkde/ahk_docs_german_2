Nummer

Konvertiert eine numerische Zeichenkette in einen reinen Integer oder eine reine Floating-Point-Zahl.

Zahl := Number(Wert)

Rückgabewert

Typ: Integer oder Float

Diese Funktion gibt das Ergebnis der Konvertierung von Wert in einen reinen Integer oder eine reine Floating-Point-Zahl, oder Wert selbst, wenn es bereits ein Integer- oder Float-Wert ist, zurück.

Bemerkungen

Wenn der Wert nicht konvertiert werden kann, wird ein TypeError ausgelöst.

Um festzustellen, ob ein Wert in eine Zahl umgewandelt werden kann, nutzen Sie die IsNumber-Funktion.

Number ist eigentlich eine Klasse, kann aber als Funktion aufgerufen werden. Mit Wert is Number kann geprüft werden, ob ein Wert eine reine Zahl ist.
