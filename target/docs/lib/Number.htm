Number

Wandelt eine numerische Zeichenkette in einen reinen Integer oder eine reine Floating-Point-Zahl um.

Zahl := Number(Wert)

Rückgabewert

Typ: Integer oder Float

Diese Funktion gibt das Ergebnis der Umwandlung von Wert in einen reinen Integer oder eine reine Floating-Point-Zahl zurück, oder Wert selbst, wenn es bereits ein Integer- oder Float-Wert ist.

Bemerkungen

Wenn der Wert nicht umgewandelt werden kann, wird ein TypeError ausgelöst.

Mit der IsNumber-Funktion kann ermittelt werden, ob ein Wert in eine Zahl umgewandelt werden kann.

Number ist eigentlich eine Klasse, kann aber als Funktion aufgerufen werden. Mit Wert is Number kann geprüft werden, ob ein Wert eine reine Zahl ist.
