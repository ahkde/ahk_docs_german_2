DriveGetSerial

Gibt die Datenträgerseriennummer eines bestimmten Laufwerks zurück.

Nummer := DriveGetSerial(Laufwerk)

Parameter

Laufwerk

Typ: Zeichenkette

Der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und einem umgekehrten Schrägstrich, oder ein Netzwerkpfad im UNC-Format wie \server1\share1.

Rückgabewert

Typ: Integer

Diese Funktion gibt die Datenträgerseriennummer des angegebenen Laufwerks zurück. Siehe Format, wie man diesen Wert in einen hexadezimalen Wert umwandeln kann.

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst.
